# Technovation Submission Checklist

## Required Files
- [ ] Anonymized manuscript (manuscript_anonymous.tex -> compile to PDF)
- [ ] Title page (title_page.pdf -- separate file with authors, affiliations, ORCID)
- [ ] Highlights (highlights.txt -- 5 bullets, each <=85 characters)
- [ ] Cover letter (cover_letter.pdf)
- [ ] Graphical abstract (graphical_abstract.png -- min 531x1328 px)
- [ ] Figures (separate folder, PDF format, 300+ DPI)
- [ ] References (references.bib)

## Content Requirements
- [ ] Abstract <=200 words (currently: 181)
- [ ] Body <=10,000 words (target: 9,500-10,000)
- [ ] Keywords: 1-7 (currently: 7)
- [ ] Reference style: author-year Harvard (elsarticle-harv.bst)
- [ ] Sections numbered: 1., 1.1, 1.1.1
- [ ] All figures/tables cited in text
- [ ] No undefined references

## Double-Blind Requirements
- [ ] No author names in manuscript
- [ ] No affiliations in manuscript
- [ ] No acknowledgements in manuscript
- [ ] No self-identifying citations
- [ ] File metadata anonymized (no author names in PDF properties)
- [ ] GitHub URL anonymized

## Elsevier Required Statements
- [ ] CRediT authorship contribution statement (in title page)
- [ ] Declaration of competing interest
- [ ] Data availability statement
- [ ] Funding information (in title page)

## Pre-Submission Checks
- [ ] Spell check completed
- [ ] All equations numbered and referenced
- [ ] Figure captions descriptive and self-contained
- [ ] Table notes explain all abbreviations
- [ ] Reference list alphabetical by first author
- [ ] DOIs included where available
